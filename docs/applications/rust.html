Adding Numbers

Install Rust by following the instructions at https://rustup.rs/. Test your installation: 

echo 'fn main() { println!("Hi"); }' > tmp.rs
rustc tmp.rs
./tmp

The program should run and print ‘Hi’.

Now let's write a program that adds two numbers and prints their sum. 

use std::io;
use std::io::BufRead;

fn main() {
    let stdin = io::stdin();

    for line in stdin.lock().lines() {
        let s = line.expect("A");
        let mut sum = 0;
        for w in s.split_whitespace() {
            let x : i32 = w.parse().expect("B");
            sum += x;
        }
        println!("{}", sum);
    }
}

The calls to expect make this program fail hard if something goes wrong.
