Programming Languages:
Applications and Design

This module has two halves.

  1. Applications: we solve five problems in five languages, and we discuss how different solutions to the same problem compare.
  2. Design: we cover basics of programming language theory, building on the examples from the first half

The assessment has three components.

  1. Classes: you will solve two problems in each of the five languages
  2. Programming Homework: you will implement a dependency manager
  3. Essay: you will write $\sim2000$ words about a topic related to programming languages, and you will present your work in front of your colleagues
