Homework: Dependencies Manager

Most operating systems and most programming languages come with tools for installing and uninstalling. In the case of operating systems, one installs and uninstalls ‘packages’ or ‘apps’; in the case of programming languages, libraries. For example, MacOS has homebrew, Debian and Ubuntu have apt, Haskell has cabal, Rust has ?, OCaml has opam, Python has pip. Your task is to implement the basic functionality of such a dependency manager.

Here is an example input: 

Package: A
Provides: D

Package: B
Provides: D

Package: C
Provides: D
